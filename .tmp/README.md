# .tmp (Temporary Workspace)

This folder contains intermediate files, test data, and temporary outputs.

## Purpose
- Store scraped data before processing
- Save test outputs during development
- Keep logs during debugging

## Important
**All files in this folder are ephemeral and can be deleted.**

The "real" deliverables live in the cloud (Google Sheets, Notion, Firebase, etc.).

## .gitignore
This folder should be ignored by Git to avoid committing temporary data.
