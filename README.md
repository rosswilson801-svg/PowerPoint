# Clarity Website

A premium, localized curriculum platform for English-medium schools globally. 

## Features

- **Global Regional Content**: Localized PSHE and pastoral content for various territories (UK, US, International).
- **Dual Flow**: Instant access to pre-built subscription packages or bespoke custom curriculum orders.
- **Lead Capture**: Integrated lead magnet (sample PSHE lesson) and detailed inquiry forms.
- **Admin Insights**: Secure, password-protected dashboard linked to Supabase for managing inquiries and leads.
- **Modern UI**: Built with React, Tailwind CSS, and Framer Motion for a high-end, responsive experience.

## Tech Stack

- **Frontend**: React 19, Vite, Framer Motion, Lucide Icons, Tailwind CSS
- **Backend/Auth**: Supabase (Database, Auth, Row Level Security)

## Getting Started

### Prerequisites

- Node.js (v18+)
- npm

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

### Deployment

This project is ready to be deployed to **Vercel** or **Netlify**. 
1. Connect your GitHub repository.
2. The build command is `npm run build`.
3. The output directory is `dist`.
